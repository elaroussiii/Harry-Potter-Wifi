Harry Potter Wiki – Clone (Projet Web)

Ce projet est une reproduction du design de la page d’accueil du Harry Potter Wiki (Fandom).
L’objectif était de recréer la mise en page, la barre latérale, l’en-tête, la zone héro et le panneau de contenu en suivant la structure du site original.

Technologies utilisées

HTML5

CSS3

Organisation du projet avec un dossier pictures/ pour les images et icônes.

Fonctionnalités

Header fixe comme sur Fandom

Sidebar verticale avec icônes

Zone hero avec image de fond et menu

Panel blanc en bas avec layout en deux colonnes

Bloc “About Us” stylisé

Mise en page responsive basique

Objectif pédagogique

Ce projet a été réalisé dans le cadre d’un exercice permettant :

de pratiquer le HTML/CSS avancé

d’apprendre la reproduction de maquettes web

de travailler l’organisation d’un projet web

d’utiliser Git et GitHub pour versionner et publier le projet
