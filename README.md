# Harry Potter Wiki – (Projet Web)

Ce projet est une reproduction du design de la page d’accueil du **Harry Potter Wiki (Fandom)**.  
L’objectif était de recréer la mise en page, la barre latérale, l’en-tête, la zone héro et le panneau de contenu en suivant la structure du site original.

## Technologies utilisées
- **HTML5**
- **CSS3**

## Organisation du projet
- Un dossier **pictures/** contient les images et icônes du site.  
- Fichiers principaux :
  - `index.html`
  - `style.css`

## Fonctionnalités réalisées
- Header fixe similaire à Fandom  
- Sidebar verticale avec icônes  
- Zone héro avec image de fond  
- Menu central  
- Panel blanc en deux colonnes  
- Bloc "About Us" reproduit  
- Mise en page responsive simple
